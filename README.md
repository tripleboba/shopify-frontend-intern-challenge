## Frontend Intern Shopify Challenge - Fall 2022
Framework: React <br>
Styling: Bulma framework <br>
Handle data using: State Hook, async fetch func <br>
Deployed at: https://ashleytran-shopify-frontend-challenge-fall2022.netlify.app/

### Local Setup
1. Install dependencies with `npm i`.
2. Using your own OpenAI account: creat `.env` file with this format in the file `REACT_APP_OPENAI_API_KEY='your-api-key'`
2. Update the .env file with your correct local information 
  - username: `labber` 
  - password: `labber` 
  - database: `midterm`
3. Run `npm start` -> `http://localhost:3000/`
