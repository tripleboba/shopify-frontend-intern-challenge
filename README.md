### Local Setup
1. Install dependencies with `npm i`.
2. Using your own OpenAI account: create `.env` file with this format in the file `REACT_APP_OPENAI_API_KEY='your-api-key'`
3. Run `npm start` -> visit `http://localhost:3000/`
